## Summary

**Parakeet uses a modern, layered architecture:**

1. **Frontend:** React 18 + Redux Toolkit for state management
2. **Styling:** Tailwind CSS + Ant Design for UI
3. **Backend:** Firebase (Auth, Firestore, Storage)
4. **AI:** OpenAI (Embeddings + Chat) for RAG-powered search
5. **Real-time:** Firestore listeners for instant updates
6. **Routing:** React Router DOM for client-side navigation

**The architecture enables:**

- Real-time data synchronization
- Semantic search with RAG
- Scalable state management
- Modern UI/UX
- Efficient data flow
